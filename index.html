User Guide

1. Import Data: Click Import Gradesheet and select your unofficial PDF transcript. This auto-fills your CGPA, Credits, and Name.

2. Plan Semester: Click + Add Course to estimate next semester's grades.

3. Retakes: If retaking a course, check the "Retake" box in the planner and select the old course from the dropdown. The system will accurately replace the old grade.

4. Manage Grades: Click Grades in the top nav to view imported history. You can manually delete unwanted courses.
